# Strategic Portfolio Selection: Consumption Thematic Funds

Exploratory data analysis on consumption-themed mutual funds to support 
evidence-based portfolio construction decisions.

## Overview
This project analyzes consumption thematic funds: mutual funds focused on 
consumer goods, retail, and spending sectors, using EDA techniques to 
evaluate their return profiles, risk characteristics, and suitability for 
portfolio inclusion. Findings are presented in a structured report with 
visualizations.

## Tech Stack
- **Python** – Pandas, NumPy, Matplotlib, Seaborn
- **Jupyter Notebook** – Analysis and visualization
- **Excel** – Raw data source (`data.xlsx`)

## What Was Analyzed
- Return distributions and statistical summaries across multiple funds
- Correlation analysis between funds to assess diversification benefit
- Risk-return tradeoff visualization to guide portfolio selection
- Comparative EDA across fund categories within the consumption theme

## Files
- `report code.ipynb` – Full analysis notebook
- `report.pdf` – Written findings and visualizations
- `presentation.pdf` – Summary slide deck
- `data.xlsx` – Raw fund data

## Key Takeaway
Identified funds with favorable risk-adjusted return profiles within the 
consumption theme, with correlation analysis highlighting diversification 
opportunities.
